# Expense Tracker Collab Project

- Track expenses program allow user to organize spendings

## Roles and Tasks

- Hansel: Responsible for Handling Expenses

- Dustin: Responsible for calculating the total cost of the expenses

- Matthew: Responsible for filtering Expense categories

## Expense Tracker Features

- Allows user to input Category, Item, Cost, and Date

- User can Add or Remove expenses, and view them conveniently

- Filtering expenses by Category and Calculating the Total Expenses are the special features of the program

- In a nutshell, this tracker is a convenient tool to manipulate expenses
